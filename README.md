# CivicAlert

A simple web app to report and track everyday civic issues in one division by local body

## 🌐 Live
👉 https://civicalert-umber.vercel.app/

---

## 🧩 Features

* Report civic issues (Garbage, Water, Electricity, etc.)
* Category-based filtering
* Status tracking
* Cloud storage
* Mobile-friendly UI 📱

---

## Languages used

* HTML
* CSS 
* JavaScript
  
---

## 🔮 Future Improvements

* Image upload support
* Map-based reporting
* Complaint history tracking
* Notification support if in  nearby location
